# Vein-Server-Manager
A dedicated server manager for the game Vein. Auto-updates, backups, and easy configuration.

ğŸ¤– Discord Remote Control (New in v4.8)
Manage your server without logging into the host PC!
Get a Bot Token from the Discord Developer Portal.
Get your Channel ID (Right-click a channel in Discord -> Copy ID).
Paste them into the Manager's "Integrations" tab.
Commands:
!status : Check if server is Online/Offline + RAM usage.
!restart : Graceful reboot (warns players -> saves -> restarts).
!backup :Create a zip backup instantly.
(Note: The bot runs inside the Manager app. The Manager must be open for commands to work.)
